# Product Images

This directory contains product images for the GreenLife Vue application.

## Image Requirements

- All images should be in JPEG format with WebP alternatives
- Recommended size: 800x800 pixels
- File naming convention: lowercase with hyphens (e.g., vivienne-westwood-tshirt.jpg)
